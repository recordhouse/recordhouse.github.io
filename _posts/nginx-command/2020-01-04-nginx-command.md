| 명령어 | 동작 |
|:---|:---|
| # systemctl status nginx | 상태 확인 |
| # sudo systemctl stop nginx | 웹서버 정지 |
| # sudo systemctl start nginx | 웹서버 재시작 |
| # sudo systemctl reload nginx | 설정 리로드 |
