# 게임 설명
* 시작 버튼을 눌러 게임 시작
* 게임을 한판 할 때 마다 하단의 코인이 하나씩 소모됨
* 룰렛 주변으로 게임 결과가 표시 됨
* 승리할 경우 랜덤으로 코인이 지급
* 짱깸뽀 사운드를 추가해야 하는데 구하기가 쉽지 않음
